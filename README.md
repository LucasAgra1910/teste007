# Subir uma alteração
Fazer uma alteração no git bash da pasta do seu repositório rodar os seguintes comandos

1 git status

2 git add .

3 git status

4 git commit -m "NOME DO COMMIT"

5 git push

OBS Caso precise configurar:

git config --global user.email "you@example.com"

git config --global user.name "Your Name"

Automáticamente ele vai abrir uma caixinha pode confirmar!

# Branchs
Criar uma branch na plataforma do github EX: 'dev'
